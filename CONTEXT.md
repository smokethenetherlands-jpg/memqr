# Страница памяти — Левкина Валентина Ивановна

## Что за проект

Цифровая страница памяти прабабушки — статический HTML/CSS/JS без бэкенда.
Файл: `v6-cinematic.html` (рабочая версия, тёмный кинематографичный стиль, золото на чёрном).
Шрифты: Cormorant Garamond + Inter.

Цель: личное использование + прототип для коммерческого продукта (QR-таблички на могилы).
